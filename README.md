# LED Control Circuit
Project Number: 2  
Student Name(s): Romandeep Singh, Sangam, Harleen Kaur

## Description
This project creates a Christmas-style LED blinking sequence using an Arduino Uno, 6 LEDs, resistors, and Tinkercad simulation.

## Features
- 6 LEDs blinking one after another  
- Custom delay time  
- Breadboard + Arduino wiring
